PacAssets 使用 [`h5ai`](https://larsjung.de/h5ai/) 构建，h5ai 是用于 HTTP Web 服务器的现代文件索引器，专注于文件。目录以吸引人的方式显示，浏览它们会通过不同的视图、面包屑和树视图得到增强。正因为 h5ai 是文件 '索引器'，所以 PacAssets 不支持客户端上传文件。但是这不影响它作为一个分享用的下载站点。

## 目录结构

PacAssets 中分类规则和逻辑

- ChatSheet - 该文件夹内存放各种编程语言的速查表(类似于一个简要大纲或者说是简要指令表)
- CookBook - 包含各种编程语言/英语/...等的 PDF 指南(电子书)
- Image - 图片资源
- Maths - 包含了研究生数学建模相关的资料
- Software - 常用/实用/好用的软件(大部分都使用过魔法 🧙‍♂️)，因本人使用 MacOS，故 Mac 软件现在较多，Win 版软件后续会大量补充
- Typora-theme - 桌面端 Markdown 编辑器 [`Typora`](https://typora.io/) 好看实用的主题整理[^1]
- Video - `Youtube` 视频资源

[^1]: 现阶段 `Typora` 不再免费，您可以到 [`Software`](https://downloads.crybabyaq.love/Software/) 中选择您的平台，下载被魔法 🧙‍♂️ 过后的 Typora，或者您直接使用 PacDocs 集成的 `PacDown`，点击侧边工具栏呼出就能编写 Markdown
